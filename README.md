
			Основные функции
1. Объединение данных
Соединяет полигональный слой izuch_A с CSV-слоем izuch по полю ukID
Создает новый слой в памяти с объединенными атрибутами
Сохраняет результат в GPKG-файл

2. Проверка геометрии
Выполняет проверку геометрии и исправляет её автоматически

3. Очистка атрибутов
Удаляет ненужные поля
Сохраняет только релевантные данные

4. Сохранение результатов
Сохраняет промежуточные и финальные результаты в GPKG-формат
Добавляет обработанные слои в проект

Входные данные
Векторный слой: izuch_A
CSV-слой: izuch

Поле для объединения: ukID

Выходные файлы
izuch_joined.gpkg - промежуточный слой с объединенными данными
izuch_joined_clean.gpkg - очищенный слой после удаления полей
izuch_fixed.gpkg - слой с исправленной геометрией (если были ошибки)

			Удаляемые поля
Скрипт автоматически удаляет следующие поля:
	- field_14
	- IDотчетаРГФ
	- nom_1000
	- name_otch
	- god_end
	- god_nach
	- in_n_tgf
	- in_n_rosg
	- tgf
	- vid_name

			Система координат
Все выходные файлы сохраняются в системе координат Pulkovo 1942 (EPSG:4284)

			Порядок выполнения
Загрузка исходных слоев из проекта
Объединение данных по ключевому полю
Проверка и исправление геометрии
Очистка атрибутов от ненужных полей
Сохранение результатов в GPKG-файлы
Добавление финальных слоев в проект QGIS

			Требования
QGIS 3.x с установленными модулями processing
Исходные слои должны быть загружены в проект

			Использование
Загрузите слои izuch_A и izuch в проект
Запустите скрипт из консоли Python в QGIS
Результаты автоматически появятся в проекте и сохранятся на диск
